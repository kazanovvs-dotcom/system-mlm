# Закладка по банкам контента

Мы остановились после следующих этапов:

1. Собрали универсальный шаблон банка из 100 тем для разных MLM-ниш.
2. Собрали первый готовый банк:
   `100 тем для рилс для ниши БАДы / wellness`, направленных на обычных клиентов.

Следующая точка возврата:

- адаптировать банк `БАДы / wellness` под конкретные продуктовые подниши:
  - коллаген
  - магний / антистресс
  - энергия / ресурс
  - ЖКТ
  - иммунитет
- затем сделать следующие банки:
  - товары для дома
  - косметика и уход
  - похудение / energy

Когда вернемся к этой ветке, логичный следующий шаг:

`Собрать 100 тем для конкретного продукта внутри БАДов`

Рекомендуемый первый вариант:

`100 тем для рилс для продукта "коллаген"`
